# Customer Shopping Analysis Dashboard

An interactive Business Intelligence dashboard project built using **MySQL** and **Power BI** to analyze customer shopping behavior, revenue trends, product performance, and subscription insights.

## Project Overview

This project focuses on transforming raw shopping data into meaningful business insights using SQL queries and interactive dashboard visualizations. The dashboard helps analyze customer purchasing patterns, top-selling products, revenue contribution, subscription behavior, and discount usage.

## Tools & Technologies Used

* MySQL
* Power BI
* SQL
* DAX
* Data Visualization

## Key Features

* Revenue analysis by gender and age group
* Top-selling products within each category
* Customer segmentation (New, Returning, Loyal)
* Subscription vs non-subscription spending analysis
* Discount and promo code usage analysis
* Interactive KPI cards, charts, slicers, and filters
* Dynamic dashboard visuals using Power BI

## Business Questions Solved

* Which customer groups generate the highest revenue?
* Which products are most purchased in each category?
* Do subscribed customers spend more?
* Which products are heavily purchased using discounts?
* Are repeat buyers more likely to subscribe?
* Which age group contributes most to revenue?

## Dashboard Insights

The dashboard provides valuable insights into customer behavior and product performance, helping businesses make data-driven decisions related to marketing, customer retention, and inventory planning.
